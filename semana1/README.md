## Resumo dos Conteúdos de HTML - Semana 1

Nesta semana, foram estudados os principais fundamentos do HTML, abordando os seguintes tópicos:

### Estrutura Básica do HTML
O HTML utiliza uma estrutura hierárquica composta pelos elementos `<html>`, `<head>` e `<body>`. O `<html>` delimita todo o documento, o `<head>` reúne informações como título, metadados, links para estilos e scripts, enquanto o `<body>` contém o conteúdo visível da página.

### Principais Tags
Entre as tags essenciais estão:
- `<h1>` a `<h6>`: Definem títulos e subtítulos, organizando o conteúdo em níveis hierárquicos.
- `<p>`: Utilizada para parágrafos de texto.
- `<a>`: Cria links para navegação entre páginas ou recursos externos.
- `<img>`: Insere imagens, sendo necessário especificar atributos como `src` (caminho da imagem) e `alt` (texto alternativo).
- `<ul>`, `<ol>`, `<li>`: Permitem a criação de listas ordenadas e não ordenadas.

### Organização do Conteúdo
O conteúdo pode ser organizado por meio de divisões (`<div>`) e seções semânticas, como `<header>`, `<main>`, `<footer>`, `<section>` e `<article>`. Essas tags facilitam a compreensão do código, melhoram a acessibilidade e contribuem para o SEO.

### Semântica e Boas Práticas
O uso de tags semânticas é fundamental para garantir que o conteúdo seja interpretado corretamente por navegadores, motores de busca e tecnologias assistivas. Boas práticas incluem:
- Utilizar títulos de forma hierárquica e adequada.
- Escrever código limpo, indentado e bem estruturado.
- Adicionar textos alternativos em imagens para acessibilidade.
- Utilizar listas para agrupar informações relacionadas.
- Priorizar a clareza e a organização do conteúdo.

Esses conceitos são essenciais para o desenvolvimento de páginas web funcionais, acessíveis e bem estruturadas.
